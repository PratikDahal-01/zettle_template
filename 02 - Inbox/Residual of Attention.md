# Residual of Attention 
This is a phenomena in which switching tasks has an effect of residue in the previous task in it
>[!note] [[Sonke Ahrens - How to Take Smart Note]] Page 18-19  says this most fittingly:
Everything needs to be taken care of else the neglected bit will nag us, this nagging will continue till the un-important bits become important here
